Configuración de Red de Área Local (LAN)

Este documento sirve como guía para la configuración y mantenimiento de nuestra infraestructura de red local, incluyendo direccionamiento IP, segmentación y políticas de seguridad.

1. Topología de Red General

Nuestra red sigue una topología estrella extendida, con un switch core central y switches de acceso distribuidos por departamento.

  • Core Switch: Cisco Catalyst 9500 Series
  • Access Switches: Cisco Catalyst 2960 Series
  • Firewall Perimetral: FortiGate 100F
  • Puntos de Acceso WiFi: Ubiquiti UniFi UAP-AC-PRO

2. Esquema de Direccionamiento IP

Utilizamos direccionamiento IPv4 privado con segmentación mediante VLANs.

2.1. Plan de VLANs

VLAN ID Nombre Subred Gateway Descripción
10 Administración 192.168.10.0/24 192.168.10.1 Servidores y gestión de red
20 Marketing 192.168.20.0/24 192.168.20.1 Departamento de Marketing
30 Desarrollo 192.168.30.0/23 192.168.30.1 Equipos de Desarrollo y Pruebas
40 Invitados (WiFi) 172.16.0.0/24 172.16.0.1 Acceso limitado a Internet para invitados
99 Voz sobre IP 192.168.99.0/24 192.168.99.1 Teléfonos IP

El servidor DHCP principal reside en el Core Switch para las VLANs de datos.

  • Rango de Exclusión: Primeras 10 IPs de cada subred reservadas para IPs estáticas.
  • Tiempo de Concesión: 8 horas para redes cableadas, 1 hora para WiFi de invitados.

2.2. Direcciones IP Estáticas Notables

  • Servidor de Archivos (NAS): 192.168.10.5
  • Servidor de Impresión: 192.168.10.6
  • Controlador de Dominio: 192.168.10.10

3. Políticas de Seguridad de Red

La seguridad es primordial. Se aplican las siguientes políticas:

3.1.1. Firewall

  • Reglas de denegación por defecto (Default Deny).
  • Inspección profunda de paquetes (DPI) para tráfico entrante y saliente.
  • Geobloqueo de IPs de regiones no deseadas.

3.1.2. Acceso a Switches

  • Acceso SSH únicamente (Telnet deshabilitado).
  • Listas de Control de Acceso (ACLs) para gestión.
  • Port-security habilitado en puertos de acceso.

  • Red Corporativa: WPA3-Enterprise con autenticación RADIUS.
  • Red de Invitados: WPA2-Personal con portal cautivo y aislamiento de clientes.

4. Mantenimiento y Monitorización

Se realizarán tareas de mantenimiento preventivo y monitorización continua.

  • Backup de Configuraciones: Diario para Firewall y Switches Core.
  • Monitorización de Ancho de Banda: PRTG Network Monitor.
  • Sistema de Detección de Intrusos (IDS): Snort en el firewall.
